HOW TO USE GIT

First:
	set user name and email

	git config user.name XXXXX
	git config user.email XXXXX@XXX.com

Second:
	clone or init a git project

	(1)clone
	git clone https://github.com/XXXXX/XXXXX
	(2)init
	mkdir project
	cd prongject
	git init

Third:
	add and commit file into git project
	
	(1)add
	git add file
	(2)commit
	git commit file -m "P.S."
	
Fourth:
	get git project info

	git status
